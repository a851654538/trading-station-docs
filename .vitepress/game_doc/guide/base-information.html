Skins77饰品开放平台 是什么？ ​

Skins77饰品开放平台是一个交易平台，支持饰品自动化快捷采购。

前提条件 ​

接入方需要准备以下信息

  • 获取Skins77账号

    后续如果迭代了购买功能, 您需要预充值账户余额, 才能完成购买饰品的操作, 需要您自行通过接口进行余额查询, 以及进行充值, 保证账户余额充足, 以免影响使用。

  • appId

    appId: 所有接口都需要在 body 中填写您的 appId

  • appSecret

    appSecret: 所有接口都需要在 body 中填写您的 appSecret

请求方式 ​

POST 方式发起请求。

请求参数 ​

请求体类型 json 在发起请求时，请求体中可能会包含两类参数：公共请求参数和接口特有的业务参数。

  • 公共参数
参数名参数值说明
appId1862049593660026880appId
appSecrete10adc3949ba59abbe56e057f20f883eappSecret
t1734599556000毫秒时间戳
  • 接口特有的业务参数是各个接口特有的，参考各接口的参数描述。

字符编码 ​

请求及返回结果使用 UTF-8 的字符集进行编码。

签名机制 ​

目前无

返回结果 ​

API请求返回以下结果

调用成功 ​

json5
{
    "code": 200,
    "data": {
        "money": "0.02"
    },
    "msg": "success"
}

调用失败 ​

json5
{
    "code": 500,
    "msg": "fail"
}

字段说明 ​

字段类型是否一定返回说明
codeInteger是本次请求是否成功,200代表成功, 500代表失败｜
msgString是信息｜
